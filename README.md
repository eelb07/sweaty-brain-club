## ✅ Commit Message Rules ✍️

**📄 `docs: Update README`**

- 리드미 수정 시 사용

**Examples:**
- `docs: 📄 Update README with commit rules`
- `docs: 📝 Add study guide to README`
- `docs: 🔧 Fix formatting in README`

***
**🧠 `feat: Add [문제 번호]`**

- 알고리즘 코드 업로드 시 사용

**Examples:**
- `feat: 🧠 Add BOJ 10828 stack solution`
- `feat: 🌟 Add Programmers 1114 BFS solution`
