# [14501] 퇴사

## 🔗 Problem Link  
https://www.acmicpc.net/problem/14501

## 💡 Approach  
- 완전 탐색 + BFS

## 🧾 Code  
```python
N = int(input())
T = []
P = []

for _ in range(N):
    t, p = map(int, input().split())
    T.append(t)
    P.append(p)

answer = 0

def dfs(day, total):
    global answer
    # print(f"day={day}, total={total}") 

    if day >= N:
        answer = max(answer, total)
        return 
    
    if day + T[day] <= N:
        dfs(day + T[day], total + P[day]) # 더해서 인자로 넘기기 

    # print("if 문을 나옴")
    # print(f"total : {total}")

    dfs(day + 1, total)

dfs(0,0)
print(answer)
```

## 🎯 Key Point  
- 완전 탐색을 DFS로 구현. 
- answer의 값을 갱신시키고 리턴 값 없이 함수를 종료 시키는 구현 

## 📚 What I Learned  
- 0번을 건너뛰는 로직이 헷갈렸다. 
- 보면 상담함. 의 내부 재귀를 다 거치고 나와서 상담안함의 분기가 시작된다. 
> ✅ 재귀 함수 안에서 호출된 함수(자식)는, 호출한 함수(부모)의 인자값을 그대로 유지한 채,
> 자기만의 새로운 인자값을 갖고 실행된다. 그리고 실행이 끝나면 그 부모 함수로 되돌아간다. ⭐️⭐️⭐️⭐️
